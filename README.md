# kanban

## Installation

### Requirements

* [Python](https://www.python.org/) 3.12
* [PipEnv](https://pipenv.pypa.io/en/latest/)

### Setup and run

To install python packages:

```
$ pipenv install
```

Run migrations:

```
$ pipenv run ./manage.py migrate
```

Then run the development server:

```
$ pipenv run ./manage.py runserver
```
